# Snowball Thrower (Winter25-Hackathon)

Small Phaser3 game with optional webcam hand-tracking (MediaPipe) and multiplayer support.

## Features
- Single-player vs AI and simple multiplayer via Network module
- Hand-tracking throw detection (MediaPipe HandLandmarker) — optional
- Sprite-based player with directional walk animations
- Snowball physics, health/charge bars, rematch flow
